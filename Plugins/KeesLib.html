A simple library for my plugins!

    This is a library I use for my plugins to make my life easier. But you can use it too! How? Just take a look!
    It contains a few methos for checking bukkit versions and if certain dependencies are enabled on the server. It also has an amazing UpdateChecker!


    MAVEN:

  1. Add the JitPack repository to your build file 
    
								<repositories>
									<repository>
										<id>jitpack.io</id>
										<url>https://jitpack.io</url>
									</repository>
								</repositories>
  2. Add the dependency 
    
								<dependency>
									<groupId>dev.casiebarie</groupId>
									<artifactId>keeslib</artifactId>
									<version>VERSION</version>
								</dependency>

    GRADLE:

  1. Add the JitPack repository to your build file 
    
								allprojects {
									repositories {
										...
										maven {url 'https://jitpack.io'}
									}
								}
  2. Add the dependency 
    
								dependencies {
									...
									implementation 'dev.casiebarie:keeslib:VERSION'
								}
